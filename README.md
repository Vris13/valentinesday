A romantic Greek Valentine's Day proposal webapp.

## Features
- Interactive Yes/No question system
- Persistent "No" responses with increasing plea
- Responsive design for mobile and desktop
- Beautiful pink gradient background
- Date details reveal on "Yes"

## Deploy to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel`
3. Follow the prompts to deploy

Or deploy via Vercel dashboard by connecting your GitHub repository.

## Local Testing
Simply open `index.html` in your browser.
